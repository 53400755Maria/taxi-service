# Taxi Service

## Описание

Веб-приложение для заказа такси (SPA). Позволяет просматривать список доступных водителей, создавать заказы, отслеживать их статус и управлять заказами.

**Реализованные endpoints:**
- `GET /` — главная страница (UI)
- `GET /api/drivers` — список водителей
- `GET /api/orders` — список заказов
- `POST /api/orders` — создание нового заказа
- `PUT /api/orders/<id>` — обновление статуса заказа
- `GET /api/health` — проверка работоспособности (health check)

## Технологии

- **Python 3.11** — язык программирования
- **Flask** — веб-фреймворк
- **pytest** — тестирование
- **flake8**, **black** — линтеры и форматирование кода
- **Docker**, **docker-compose** — контейнеризация
- **GitHub Actions** — CI/CD (непрерывная интеграция)
- **Render** — облачная платформа для развертывания (CD)
